# Users

## Objectif

Le système Users permet de gérer toutes les personnes utilisant VTA Software Suite.

---

## Un utilisateur possède

- Nom
- Prénom
- Email
- Téléphone
- Photo
- Mot de passe
- Langue
- Fuseau horaire
- Statut

---

## Statuts possibles

- Actif
- Inactif
- Suspendu
- En attente d'invitation

---

## Un utilisateur peut

- appartenir à plusieurs entreprises
- avoir plusieurs rôles
- changer d'entreprise
- modifier son profil
- modifier son mot de passe

---

## Sécurité

Chaque utilisateur possède un identifiant unique.

Les mots de passe ne sont jamais enregistrés en clair.

Toutes les connexions sont enregistrées dans les Audit Logs.
