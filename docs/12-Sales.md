# Sales

## Objectif

Le module Sales permet de gérer toutes les ventes de l'entreprise.

Il complète le module POS en offrant une gestion complète du cycle de vente.

---

# Fonctionnalités

- Création de devis
- Commandes clients
- Facturation
- Livraison
- Retours clients
- Notes de crédit
- Historique des ventes

---

# Clients

Chaque client possède :

- Code
- Nom
- Téléphone
- Email
- Adresse
- Crédit autorisé
- Solde
- Historique des achats

---

# Documents

Le système gère :

- Devis
- Commande
- Facture
- Retour
- Note de crédit

---

# Statuts

- Brouillon
- Confirmé
- Livré
- Facturé
- Annulé

---

# Paiements

- Espèces
- Carte bancaire
- Mobile Money
- Virement bancaire
- Paiement partiel
- Paiement complet

---

# Intégrations

Le module Sales communique avec :

- Inventory
- Accounting
- CRM
- Audit Logs

---

# Règles

Chaque vente doit mettre à jour automatiquement :

- le stock
- le chiffre d'affaires
- les comptes clients
- les Audit Logs
